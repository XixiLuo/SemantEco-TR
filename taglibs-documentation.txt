
SemantEco provides a JSP tag library that simplifies the generation of the main page by condensing references to the various scripts, stylesheets, and facets provided by the different modules into three separate tags: <module:scripts />, <module:stylesheets />, and <module:facets />. The implementations of the tag handlers for these tags are located in the edu.rpi.tw.escience.semanteco.taglib package. For a complete overview of JSP tag handlers see [1].

[1] Oracle. Custom Tags in JSP Pages. 2010. In The Java 5 EE Tutorial. http://docs.oracle.com/javaee/5/tutorial/doc/bnalj.html. Last accessed 2013-02-23.
